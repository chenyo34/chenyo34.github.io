Hidden Markov Model

Posted on 2023-01-17 Edited on 2023-01-23

Background

Hidden Markov Model is one of the Dynamic Model, which is a subclass of machine learning based on Bayesian Theory.

Here we need to determine some important notations before moving forward.()

  • Time Sequence:
    $T = {0, 1, …, N}$,$\forall t \in T$, $t$ is an non-negative integar.

  • Observation Variable Set and Sequence:

    Feasible Set for Observation:
    $V = {v_0, v_1, …, v_N}$

    Obervation Sequence:
    $O = {o_0, …, o_N}$, where $o_i$ represents the observation in $i$ time.

  • Status Variabel Set and Sequence:

    Feasible Set for Stataus:
    $Q={q_0, q_1, …, q_n }$

    Status Seqence:
    $S={s_1, …, s_N}$, where $s_i$ is the $i^{th}$ status in the feasible set.

  • Transmision Matrix:

    A is a $n\times n$ Matrix donated as $\mathbf{A = [a_{ij}]}$, here $a_{ij}$ is the probability of a transmission from status $q_i$ to $q_j$, where $q_i,q_j \in Q$. It can be definied as $a_{ij} = P(a_{t+1} = q_j| a_t = q_i)$, $t \in T$ and $t < N$.

  • Emission Matrix:

    B is a $n\times m$ Matrix donated as $\mathbf{B = [b_{j}(k)]}$, here $b_{j}(k)$ is the probablity of the status $q_j$ emit an observation $o_k$, where $q_j \in Q$ and $o_k \in V$

What assumptions/scenarios are required

  • The Markov Assumption
    The next status is dependent on the current status.
    $$
    P(s_{t+1}|{s_1,o_2},{s_2,o_2},…,{s_t, o_t})=P(s_{t+1}|s_t)
    $$
  • The stationarity Assumption
    The state transition probabilities are independent of the actual time at when the transitions take place.
    $$
    P(q_{t_1+1} = j|q_{t_1} = i)=P(q_{t_2+1} = j|q_{t_2} = i)
    $$
  • The output Assumption
    The current observation is conditional on the current status.
    $$
    P(o_t|{o_1, …, o_{t-1}}, {s_1,…,s_t}) = P(o_t|s_t)
    $$
    , which implies that
    $$
    P(O_{1, .., T}) = \sum^T_{t=1} P(o_t|s_t).
    $$

AMM (Automated Market Makers)

A decentralized and permissionless algorithm that : 

1. works as a market marker but in an automated way
2. is not governed by a single entity  
3. stores assets in different place  
4. anyone can use in anywhere.

More formal speaking, a set of smart contracts that define how liquidity is managed.

AMM in Uniswap

==Version 2==

Uniswap uses the Constant Funciton AMM, whose core function was formed as,
$$
x * y = k
$$

and trade functions is,

$$
(x + r\Delta x) * (y - \Delta y) = k.
$$

$\textbf k$ is unchangable and cannot be zero
$\textbf x$ refers to token 0
$\textbf y$ refers to token 1
$\Delta$ represents the difference
$\textbf r$ is the trading fee/rate, which is known and considered as a constant factor

Spot Prices
Def: The prices only reflect current market prices and calculated by the amounts of reservces of tthe tokens.
$$
P_x = \frac{y}{x}, P_y = \frac{x}{y}
$$

Trade Prices
Def: Based on the spot prices, add up the influence from supply and demand, high demand increases the price and vice versa.
$$
\Delta y = y \cdot \frac{r\Delta x}{x + r\Delta x}\
\Delta x = x \cdot \frac{\Delta y}{r(y-\Delta y)}
$$

Gausian Distribution

Posted on 2022-12-21 Edited on 2023-01-17

Gausian Distribution

1-Dimension:

$$
x \sim Gausian(\mu_{1\times 1}, \Sigma_{1\times1})
$$

$$
p(x) = \frac{1}{({2\pi})^\frac{1}{2}\sigma}exp[-\frac{(x-\mu)^2}{2 \sigma^2}]
$$

Multi-Dimension:

$$
x \sim Gausian(\mu_{1\times p}, \Sigma_{p\times p})
$$

$$
p(x) = \frac{1}{({2\pi})^\frac{p}{2}{|\Sigma|}^p}exp[-\frac{1}{2}{(x-\mu)^T\Sigma^{-1}(x-\mu)}]
$$

Maximum Likelihood Estimation(MLE):

$$
\hat\mu_{MLE,unbiased} = \frac{1}{N}\sum^N_{i=1} x_i
$$

$$
\hat\Sigma_{MLE,biased} = \frac{1}{N}\sum^N_{i=1}(x_i - \hat{\mu})^T(x_i-\hat{\mu})
$$

$$
\hat\Sigma_{MLE,unbiased} = \frac{1}{N-1}\sum^N_{i=1}(x_i - \hat{\mu})^T(x_i-\hat{\mu})
$$

If we use MLE to estimate the variance, $\hat{\sigma}$ would be $\frac{1}{N}$ smaller than $\sigma$.

Geometric Meaning:

Distane Metric:
$\Delta = (x-\mu)^T \Sigma^{-1} (x-\mu)$, which is known as Mahalanobis Distance.
$d^2 = (x-\mu)^T (x-\mu)$, which is basic Euclidean Distance.

$\Delta == d^2$ if and only if $\Sigma^{-1} = 1$.

Euclidean Distance: It assumes the data following an isotropically Gaussian Dist., each feature contribute equally.
Mahalanobis Distance: It will scale the values and measure the correlation among variables and relaxes the assumption of Euclidean Distance.

If $p >= 2$ and $\Sigma$ is postive definite mastrix($\sigma_{i,j} > 0,\forall i,j\in[1,p]$). Noticing that $\Sigma = U^T \Lambda U$,where $U = (u_1, u_2, …, u_p)_{p\times p}$ and $\Lambda = diag(\lambda_1,\dots, \lambda_p)$. Consider $y_i= (x-\mu)^Tu_i$, $\Delta = \sum^p \frac{y_i^2}{\lambda_i}$. According to the definition and formula of Elliptic geometrics, once we have an given x, we can generate a contour line on the feasible set since both $\Sigma$ and $\mu$ are considered as known parameters at this stage.

Gausian Constraint

  1. Limitation of Representive
    In many cases, it is extremely hard to capture the patterns with only one Gaussian. For example, there may exist highly data-concentrated spaces, it may require a combination of multiple Gaussian to invovle.(GMM)
  2. Computation Complexity
    In each Gaussian model, we need to compute and estimate $\frac{p(p+1)}{2}$ parameters in $\Sigma$ and $parameters$. Which as known as a $O(p^2)$ complexity.
